# Listen on a localhost

```bash
$ git clone git://github.com/59naga/npm.berabou.me.git
$ cd npm.berabou.me

$ npm install bower --global
$ npm install onefile --global
$ npm install --production
$ npm start
# Listen to http://localhost:59798/?debug
```
